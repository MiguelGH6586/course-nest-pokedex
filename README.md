# Ejecutar en desarrollo

1. Clone repository
2. Run
```
yarn install
```
3. Nest CLI Installed
```
npm i -g @nestjs/cli
```
4. Run DB
```
docker compose up -d
```

## Stack used
* MongoDB
* NestJS
* Docker
